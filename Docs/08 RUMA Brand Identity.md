# ## Logo Direction  
#   
# Logo Type:  
# Wordmark + Symbol  
#   
# Symbol Meaning:  
#   
# Community  
# Connection  
# Togetherness  
#   
# Avoid:  
#   
# Kathakali  
# Kerala map  
# Boat race  
# Coconut tree  
# Religious symbols  
#   
# The logo should remain timeless and usable for at least 10 years.  
#   
# ---  
#   
# ## Photography Style  
#   
# Use real residents whenever possible.  
#   
# Prioritize:  
#   
# Families  
# Children  
# Celebrations  
# Volunteers  
# Food  
# Games  
#   
# Avoid stock imagery.  
#

#

# Festive
