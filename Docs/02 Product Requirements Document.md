# Product Requirements Document  
## Product Name  
RUMA Events  
## Version  
V1.0  
## Primary User Groups  
## Residents  
Register for events, purchase tickets, upload payment proof, receive tickets.  
## Organizers  
Manage registrations, approve payments, monitor attendance, conduct lucky draws.  
  
## Core Features  
## Event Landing Page  
Displays:  
* Event name  
* Date  
* Time  
* Venue  
* Schedule  
* Registration CTA  
  
## Family Registration  
Collect:  
* Full Name  
* Flat Number  
* Phone Number  
* Email (Optional)  
  
## Ticket Selection  
Configurable categories:  
* Adult  
* Child (5–12)  
* Child (Below 5)  
Dynamic pricing.  
Automatic total calculation.  
  
## Payment Upload  
User uploads:  
* UPI payment screenshot  
Status:  
* Pending Approval  
  
## Admin Approval  
Admin can:  
* View submission  
* Review screenshot  
* Approve payment  
* Reject payment  
  
## Ticket Generation  
After approval:  
Generate:  
* Booking ID  
* Individual ticket IDs  
* QR codes  
* Downloadable PDF  
  
## Attendance Check-In  
Admin scans QR.  
Status changes:  
Active → Checked In  
  
## Dashboard  
Show:  
* Registrations  
* Revenue  
* Pending approvals  
* Adults count  
* Child count  
* Attendance count  
  
## Lucky Draw  
Generate coupon numbers.  
Store ownership.  
Random winner selection.
